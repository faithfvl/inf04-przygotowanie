# Arkusze praktyczne

Folder na rozwiązane arkusze egzaminu praktycznego INF.04 z poprzednich sesji.

Dla każdego arkusza tworzę osobny podfolder (np. `2024-styczen/`) i opisuję rozwiązanie według schematu z przykładu poniżej.

[arkusz-przyklad/](arkusz-przyklad/) – wzór, jak dokumentować rozwiązany arkusz

> Arkusze i informatory można pobrać ze strony Centralnej Komisji Egzaminacyjnej (CKE). Tutaj wrzucam własne rozwiązania i kod.
