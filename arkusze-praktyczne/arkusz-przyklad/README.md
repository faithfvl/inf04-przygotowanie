# Arkusz przykładowy – wzór dokumentacji

> To jest wzór pokazujący, jak opisać rozwiązany arkusz. Treść zadania jest autorska (przykładowa) i nie pochodzi z prawdziwego arkusza CKE.

## Treść zadania (skrót)
Utwórz bazę danych biblioteki oraz aplikację umożliwiającą dodawanie książek i wyszukiwanie ich po tytule.

## Użyte technologie
- Język: C#
- Baza danych: MySQL

## Schemat bazy danych
Patrz: [baza.sql](baza.sql)

## Etapy rozwiązania
1. Utworzenie bazy danych i tabel
2. Połączenie aplikacji z bazą danych
3. Implementacja dodawania i wyszukiwania książek
4. Testy działania
